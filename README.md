# Koopman-Eigenfunction Lyapunov Functions

NumPy implementations of the kernel PDE-collocation method for constructing
certified Lyapunov functions via approximate Koopman eigenfunctions, based on:

> Giesl, Hafstein, Hamzi, **Lee**, Owhadi, Santin, Vaidya (2026).
> *Kernel Methods for the Construction of Certified Lyapunov Functions
> via Approximate Koopman Eigenfunctions.*

## Method overview

For a system $\dot x = f(x)$ with a stable equilibrium at the origin and
Jacobian $E = Df(0)$, each eigenvalue $\lambda_i$ of $E$ (with $\text{Re}(\lambda_i)<0$)
yields a Koopman eigenfunction

$$\varphi_{\lambda_i}(x) = w_i^\top x + h_{\lambda_i}(x),$$

where $w_i$ is the corresponding left eigenvector and $h_{\lambda_i}$ solves the PDE

$$\nabla h(x)\cdot f(x) - \lambda_i h(x) = -w_i^\top G(x), \quad h(0)=0,\;\nabla h(0)=0,$$

with $G(x)=f(x)-Ex$ the nonlinear remainder. The Lyapunov function is

$$V^\star(x) = \sum_{i=1}^d P_{ii}\,\varphi_{\lambda_i}(x)^2, \qquad P_{ii} = -\frac{1}{2\lambda_i}>0,$$

satisfying $\dot V^\star(x) = -\sum_i \varphi_{\lambda_i}(x)^2 < 0$ for $x\ne 0$.

The PDE is solved by symmetric kernel collocation in an RKHS with anisotropic
Gaussian kernel $K(x,y)=\exp\!\bigl(-\sum_k(x_k-y_k)^2/(2\sigma_k^2)\bigr)$,
yielding a linear system of size $(n+d+1)\times(n+d+1)$.

## Quick start

```bash
pip install -r requirements.txt
cd notebooks
jupyter lab
```

The notebooks are self-contained — all kernel functions are defined inline so
they run without installing the `src/` package. The `src/kernel_lyapunov`
package provides the same implementation as a standalone importable module.
